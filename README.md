#  To-do List

Application web simple permettant de gérer une liste de tâches.

##  Fonctionnalités
- Ajouter une tâche
- Supprimer une tâche en cliquant dessus

## Technologies utilisées
- HTML
- CSS
- JavaScript

##  Aperçu
Application développée dans le cadre de mon apprentissage du développement web.
